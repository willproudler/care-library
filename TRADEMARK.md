CARE Library and CARE Machine are names associated with the Canonical Axiomatic Reduction Engine project created by William Proudler and collaborators.

A UK trademark application relating to the CARE Library name has been submitted and remains pending.

The trademark is intended to protect the identity and integrity of the open CARE commons. It is not intended to prevent legitimate study, citation, criticism, compatible implementation or non-misleading description of CARE-derived work.

Forks and compatible projects should clearly distinguish themselves from the canonical CARE Library project and must not imply official endorsement, governance or origin where none exists.

Further trademark and compatibility guidance will be developed as the project matures.
