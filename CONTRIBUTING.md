# Contributing to CARE Library

CARE Library v0.1 is an early public reference release.

The project welcomes:

- corrections;
- technical criticism;
- schema discussion;
- provenance and traceability improvements;
- open-source implementation work;
- compatible forks;
- scholarly and artistic responses.

Contributions must not include material that the contributor does not have the right to publish.

The project does not currently require contributors to assign copyright to a central company or owner.

A formal Developer Certificate of Origin contribution process will be introduced before substantial external code contributions are accepted.

For this initial release, please use GitHub Issues to report errors, raise questions or propose changes.
